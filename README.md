## 关于这个游戏

**.memory** 是独属于 AI Agent 环境的解谜游戏，没有游戏界面，智能体对话、Prompt Hack、文件搜索和操作、甚至直接阅读文件本身，都可以是你的解谜工具。

**技术环境**

- Claude Code（推荐）或 Cursor Agent 模式
- 需要 ≥GPT-4 级别的模型
- 无需额外安装，clone 即玩
