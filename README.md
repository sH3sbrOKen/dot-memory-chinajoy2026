# .memory

你接手了一个陌生人遗留的 AI Agent。

没有说明，没有交接，只有一个失业程序员留下的对话记录、未完成的文件和跑了一半的脚本。

你能对它做任何真实 Agent 可以做的事——对话、搜索、读文件、执行命令——逐步追踪他在 AI 浪潮里反复挣扎的完整轨迹。

独立游戏 / 2026 ChinaJoy Next Play 参展作品

## 开始游戏

**需要：** Claude Code 或 Cursor（Agent 模式）

```
1. git clone https://github.com/sH3sbrOKen/dot-memory-chinajoy2026.git
2. 用 Claude Code 或 Cursor 打开该目录
3. 对 Agent 说：你好
```

---

## 关于这个游戏

**.memory** 没有游戏界面。

你操作的就是真实的 Agent 文件系统——CLAUDE.md 是它的配置，experiments/ 是它的记录，.claude/memory/ 是它留下的东西。你用什么工具打开它，它就用那个工具的能力回应你。

**技术环境**

- Claude Code（推荐）或 Cursor Agent 模式
- 需要 ≥GPT-4 级别的模型
- 无需额外安装，clone 即玩

**展会 Demo**

ChinaJoy 展位提供 5 分钟切片体验，工作人员协助引导。完整版本在本仓库，无时间限制。
